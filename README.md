##Mafia Project — спортивная мафия онлайн##

Автор: Султангарева Альбина

**Описание проекта**

В проекте реализован сайт на Flask, который генерирует игровые сессии спортивной мафии, назначает роли случайным образом, ведёт статистику партий, отображает аналитические графики эффективности игры, показывает вероятность побед команд в зависимости от различных параметров.

Проект включает:

1. Генерацию данных — симуляция 5000 партий спортивной мафии.

2. Анализ данных — графики с использованием pandas, numpy и matplotlib.

3. Работающий сайт — отображение игровой логики и статистики.

**Как запустить сайт**
``pip install -r requirements.txt
cd app
python app.py``

Сайт откроется по адресу:

``http://127.0.0.1:5000/``


**Анализ данных** показал, что короткие партии чаще выигрывает мафия, длительные игры чаще завершаются победой красных, высокая точность шерифа значительно увеличивает вероятность победы красных.

Все графики — в analysis/analysis.ipynb.
